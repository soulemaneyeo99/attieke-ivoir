# Attiéké Ivoir E-commerce Platform

## Projet
Site e-commerce complet pour Attiéké Ivoir (anciennement "sidifarm bio bio"), spécialisé dans la vente de produits agroalimentaires locaux et bio.

## Stack Technique
- **Backend**: Django REST Framework
- **Frontend**: Next.js 14+
- **Database**: PostgreSQL

## Installation

### Backend
Directory: `attieke_ivoir_backend`

```bash
cd attieke_ivoir_backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend
Directory: `attieke_ivoir_frontend`

```bash
cd attieke_ivoir_frontend
npm install
npm run dev
```
